# 用户中心

### 管理员批量导入用户，校验用户信息

用户昵称 最多20个字符 可以为空 不唯一

id 导入自动分配 不考虑

性别 1 & 2 可以为空 只能是1 或 2 或 null

电话 唯一 可以为空 符合电话号码的格式

密码 6-20位 包含常见特殊字符、数字、字母 不能为空

头像 可以为空 默认不设置 让用户自己设置

创建、更改时间 导入时添加

is_delete 默认初始0 不考虑这个字段

user_status 默认正常 不考虑

邮箱 唯一 符合邮箱的格式 可以为空

账户名 唯一 不能为空 6-20位 不能包含特殊字符 只能是数字、字母、_、-

role 默认普通用户 目前只能是 0 或 1

### 更多的条件组合查询用户

用户名 账户名 性别 注册时间 权限
